# Total Time Calculator
- Used to calculate the total time in 24-hour format.

# Requirements
- python
- colorama

# How to use
```
git clone https://github.com/the-glitch69/time-calculator
sudo apt-get update
cd time-calculator
pip3 install -r requirements.txt
python time.py
```
- to make it executable
```
chmod +x time.py
```
- to run the script
```
./time.py
```
